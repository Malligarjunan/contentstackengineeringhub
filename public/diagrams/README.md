## Creating Diagrams

Recommended tools:
- **Whimsical** - https://whimsical.com (Collaborative diagrams)
- **Lucidchart** - Professional diagramming
- **Draw.io** - Free, open-source
- **Miro** - Collaborative whiteboard

## Diagram Standards

### Include in CI/CD Diagrams:
1. Source control trigger (Git webhook)
2. Build steps
3. Test stages (unit, integration, e2e)
4. Security scanning
5. Deployment stages (dev, staging, production)
6. Rollback procedures
7. Notification steps

### Export Format:
- PNG (recommended) - 1920x1080 or 16:9 aspect ratio
- High resolution (at least 1440px width)
- Transparent background preferred

### Naming Convention:
`{product-slug}-cicd.png`
